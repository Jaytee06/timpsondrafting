How residential drafting projects usually start

A helpful first step usually includes the project city and state, project type, rough size, any existing plans or photos, and whether the work is a new build, remodel, garage, ADU, or addition.

  • Share the project basics so the drafting scope is easier to understand.
  • Gather existing plans, sketches, photos, and inspiration if available.
  • Clarify timeline and permit questions before requesting a project quote.

Drafting and Permit Questions

What does Timpson need to quote a project? Project location, project type, rough size, timeline, existing plans or photos, and whether the work is a new build, remodel, garage, ADU, or addition.

What are permit-ready construction documents? Organized drawings prepared for building-department review, based on the project scope and local requirements.

How is drafting pricing scoped? Quotes are based on project size, complexity, existing information, revision needs, and permit requirements.
