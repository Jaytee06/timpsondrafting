When this service is a good fit

If you are adding square footage, building a detached garage, turning extra space into an ADU, or extending a home with a mudroom or suite, this is usually the right starting point. The drafting scope needs to reflect the existing house, the lot, and the permit path for the project.

Good starting details include the project location, rough size, whether the work attaches to the existing home, and any existing plans, sketches, or site photos.

Common project types

Detached garages and shops

Drafting support for stand-alone structures that need organized layouts, elevations, and permit-ready drawing coordination.

Home additions

Bedroom suites, mudrooms, expanded living areas, and other attached additions that need to connect cleanly to the existing structure.

ADU planning support

Accessory dwelling unit layouts that need efficient space planning and a clear documentation path before permit submission.

Permit updates and revisions

Adjustments to an existing plan package when the scope changes, comments come back, or the project needs cleaner coordination.

What helps Timpson scope the work

  • Project address or city and state
  • Approximate square footage or footprint
  • Whether the project is detached or attached to the current home
  • Existing plans, photos, sketches, or field measurements if available
  • Expected timeline and whether permit submission is part of the immediate need
