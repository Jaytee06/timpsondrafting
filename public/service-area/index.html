How remote drafting usually works

Many residential drafting jobs do not require an in-person visit at the first stage. If homeowners, contractors, or project managers can provide photos, rough dimensions, older plans, surveys, and the local permit context, Timpson can often scope custom homes, garages, additions, remodels, and permit-ready drawing packages remotely.

Remote drafting is available nationwide. On-site field measurement and nearby support are currently focused around Colorado City, Northern Arizona, and Southern Utah.

When local on-site support is the better fit

Existing homes that need field measurement

Remodels, additions, and as-built projects often benefit from a local site visit when the existing structure needs to be measured before drafting can move forward confidently.

Projects with unclear existing information

If plans, sketches, or dimensions are incomplete, nearby field verification can reduce guesswork before a drawing package is scoped.

Jurisdiction-specific coordination

Permit requirements still depend on the project city, county, and building department, even when the drafting work itself can start remotely.

Residential project types already supported

Timpson commonly works on homes, garages, ADUs, additions, remodels, as-built drawings, and permit-ready residential plan packages.

Helpful details to share first

  • Project city and state
  • Whether the project needs remote drafting only or local field measurement
  • Project type such as custom home, garage, addition, remodel, or permit update
  • Rough size, footprint, or existing-condition details
  • Any existing plans, photos, sketches, surveys, or measurements
  • Timeline and any known permit-review questions
