Remodel and As-Built Drawings

Renovation work often starts with understanding the existing structure before drafting the new direction clearly. Timpson can help document what is there now, organize remodel-ready drawings, and support the permit-prep side of residential updates.

Request a remodel quote Review permit-ready planning tips

How this work usually starts

Many remodels need existing-condition documentation before the new design can be drafted well. That is where as-built drawings can help. Once the current layout is clear, it becomes easier to organize the renovation scope, contractor coordination, and permit package.

Common remodel drawing needs

Existing-condition documentation

Measured or compiled drawings that show the current layout before the renovation scope is finalized.

Kitchen, bath, and whole-home updates

Drafting support for remodels where the existing space needs to be reworked, expanded, or clarified for construction planning.

Addition-plus-remodel coordination

Projects where a new addition connects to remodel work and both need to be represented clearly in one drafting scope.

Permit package updates

Organizing revisions and scope updates so the documents better match the work being submitted for review.

Helpful information before a quote

  • Project address and project type
  • Photos, sketches, or prior plans if any exist
  • What is staying, what is changing, and what still feels undecided
  • Whether measured existing conditions are already available
  • Expected timeline for drafting, contractor coordination, or permit review
As-built drawings are especially helpful when the current condition of the home needs to be documented before the remodel scope can be drafted with confidence.
