# Landing Page Strategy

## Node Type

Marketing-to-website strategy process.

## Purpose

This file defines how the Marketing Agent should translate paid-search and SEO evidence into landing page improvements.

## Current Surface

The active landing page lives in the frontend app. It presents Timpson Drafting & Design, captures lead details, preserves attribution, and triggers conversion tracking after a successful submission.

Agents should inspect the current source before recommending specific headline or section edits.

## Strategy Rules

Landing page changes should:

- reflect real services Timpson Drafting & Design offers,
- use high-intent search language from account data,
- preserve trust and clarity for residential drafting visitors,
- improve message match between keyword, ad, and page,
- avoid claims that need legal, licensing, pricing, guarantee, or timeline confirmation unless verified,
- preserve contact form behavior and attribution capture.

## Cross-Pollination Procedure

1. Extract the top 20 search categories or search terms from the latest weekly probe.
2. Classify each term by intent:
   - service type,
   - project type,
   - location,
   - price or timeline question,
   - permit or process question,
   - competitor or brand query,
   - irrelevant or low-intent traffic.
3. Compare high-converting terms against current landing page headings, hero copy, section headings, FAQ content, and organic SEO targets.
4. Mark each term as:
   - already prominent,
   - present but buried,
   - missing but strategically relevant,
   - not appropriate for the business.
5. Propose copy changes that improve match without stuffing keywords.
6. Identify whether the change belongs in:
   - hero headline or subheadline,
   - service section,
   - FAQ,
   - location/service area copy,
   - ad copy test,
   - organic SEO content plan.

## Landing Page Update Candidates

Typical candidate themes:

- residential drafting,
- home addition plans,
- remodel drafting,
- custom house plans,
- permit-ready drawings,
- ADU plans if offered,
- local drafting service terms,
- process clarity for first-time homeowners,
- plans-ready timeline questions.

## Approval Boundaries

The Marketing Agent may draft or implement copy that clarifies existing services.

Escalate before adding claims about:

- pricing,
- guaranteed approval,
- exact permit timelines,
- professional licensing status,
- engineering scope,
- jurisdictions served when not already documented,
- services not already offered.
